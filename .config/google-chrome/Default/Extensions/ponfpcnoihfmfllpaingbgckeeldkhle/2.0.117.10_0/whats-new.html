What's new in Enhancer for YouTube™

Important changes only compatible with the latest version of Chrome!
Open a new tab and copy/past this URL to check your version: chrome://settings/help
Restart your browser or reload alreay open YouTube pages if needed, but keep reading first

v2.0.117

Shorts

Added an option to convert Shorts so you can watch them like normal videos, and an option to completely hide Shorts. Both options are disabled by default, check the “Appearence” section of the options page to enable them.

Thanks bug

I have fixed the broken Thanks buttons located at the bottom of this page. A big thank you to all the nice people who have reported this issue...

Bugs

  • Extension partially working: update your browser to the latest version
  • Fixed the scroll issue when the video player is automatically expanded and/or when the full screen mode is active
v2.0.116

Manifest V3

Enhancer for YouTube™ is now using Manifest V3, which has been introduced to increase security, privacy, and performance of extensions. A huge code rewriting was necessary to achieve this.

Shorts

Shorts are now loaded like normal videos, exception made for shorts that appear when clicking the “Shorts” link in the main menu
Read the note for version 2.0.117 above

Dark UI

Based on your operating system's color scheme preferences, the options page is now displayed with either a light or a dark theme

Clips

Added support for clips

Playback Speed

New playback speed variation options (0.01 and 0.02) for a finer control

Volume Booster

Option to automatically boost the volume

Picture in Picture *

Show the built-in “Picture in Picture” control that is hidden by default in the video player (Chromium-based browsers only)

Bugs

Bug fixes and overall improvements for a better user experience
* Don't need the “Picture in Picture” control? No problem, use this as custom theme:#movie_player div.ytp-chrome-controls button.ytp-pip-button.ytp-button{display: none !important}
